# Resume Website

A lightweight one-page resume website built with plain HTML, CSS, and JavaScript.

## Files to edit

- `resume-data.js`
  - Replace the placeholder text with your real name, summary, experience, project details, links, and skills.
- `index.html`
  - Change the page title or structure only if you want to adjust the layout.
- `styles.css`
  - Update colors, spacing, or typography if you want a different visual style.

## Quick start

1. Open `resume-data.js`.
2. Replace the placeholder content with your real information.
3. Open `index.html` in your browser.

## Optional improvements

- Replace `resumeUrl: "#"` with a real PDF link to your resume.
- Add a profile image or headshot in the hero section.
- Deploy the site on GitHub Pages, Netlify, or Vercel.
